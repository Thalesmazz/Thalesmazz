### 🤖 IA como engenharia (não hype)

- **Produtos com LLM** — extração estruturada de documentos (PDF → JSON validado com `generateObject` + Zod), orquestração de modelos por tarefa (qualidade vs. custo) e _human-in-the-loop_ como decisão de arquitetura.
- **Agentes de código no fluxo real** — Claude Code + Cursor com protocolo estruturado (regras, anti-padrões, hooks de re-indexação). Não é "ChatGPT pontual".
- **MCP (Model Context Protocol)** — grafo de código (análise de impacto / blast radius antes de cada edição) + memória viva (ADRs, regras de negócio, restrições).
